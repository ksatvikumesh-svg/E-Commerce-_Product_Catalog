# E-Commerce Product Catalog

## Overview
This project is developed as part of Thiranex Task 5: Full-Stack Deployment & Project Architecture. It demonstrates a responsive e-commerce product catalog with product browsing, shopping cart functionality, and deployment readiness.

## Features
- Responsive User Interface
- Product Listing
- Product Details
- Shopping Cart
- Client-side Routing
- Mobile Friendly
- Optimized Assets
- Deployable to Netlify or Vercel

## Technologies Used
- HTML5
- CSS3
- JavaScript
